# DOG-HUMAN-YEARS
Convert your age in Dog Years

const myAge = 24;
let earlyYears = 2;
earlyYears *= 10.5;
//setting early years 
let laterYears = myAge -2;
//setting later years 
laterYears *= 4;
// dog years is equal to 4 time of later years.
console.log(earlyYears);
console.log(laterYears);
